🧠 Panorama des technologies IA

3.1 Les trois grandes familles d'IA

Machine Learning (ML) - L'apprentissage automatique

Le Machine Learning permet aux machines d'apprendre des patterns dans les données sans être explicitement programmées pour chaque cas.

🔍 Exemples concrets :
  • Systèmes de recommandation : Netflix, Spotify, Amazon
  • Détection de fraude : Transactions bancaires suspectes
  • Prédiction météo : Analyse de millions de données climatiques

Deep Learning (DL) - L'apprentissage profond

Une sous-catégorie du ML qui utilise des réseaux de neurones artificiels multicouches pour traiter des données complexes.

🔍 Exemples concrets :
  • Reconnaissance d'images : Google Photos, reconnaissance faciale
  • Conduite autonome : Tesla, Waymo
  • Traduction automatique : Google Translate, DeepL

Large Language Models (LLM) - Les modèles de langage

Des IA spécialisées dans la compréhension et la génération de texte, entraînées sur d'énormes quantités de données textuelles.

🔍 Exemples concrets :
  • Assistants conversationnels : ChatGPT, Claude, Bard
  • Génération de contenu : Jasper, Copy.ai
  • Résumé automatique : Notion AI, Grammarly

🦁 Safari IA : Exploration guidée

3.2 Découverte des IA de votre quotidien

Partons ensemble à la découverte des intelligences artificielles qui vous entourent déjà, souvent sans que vous le sachiez !

🎬
Netflix
Recommandation
L'algorithme de recommandation de Netflix analyse vos habitudes de visionnage, les notes que vous donnez, et même la façon dont vous naviguez pour vous proposer du contenu personnalisé.
🎯 Test pratique :
Ouvrez Netflix et observez : les vignettes, l'ordre des contenus, les catégories proposées sont uniques à votre profil !
Machine Learning Filtrage collaboratif Analyse comportementale
📸
Google Photos
Vision par ordinateur
Google Photos utilise la reconnaissance d'images pour identifier automatiquement les personnes, objets, lieux et même créer des albums thématiques.
🎯 Test pratique :
Tapez "chat", "voiture" ou "plage" dans la barre de recherche - l'IA retrouve vos photos correspondantes !
Deep Learning CNN Classification d'images
🎵
Spotify
Audio + ML
Spotify combine analyse audio (tempo, tonalité) et comportement d'écoute pour créer vos playlists "Discover Weekly" et "Daily Mix".
🎯 Test pratique :
Vérifiez votre "Discover Weekly" - 30 nouvelles chansons chaque lundi, personnalisées selon vos goûts !
Audio Analysis Collaborative Filtering NLP
📧
Gmail
Traitement du langage
Gmail utilise l'IA pour filtrer les spams, proposer des réponses automatiques ("Smart Reply") et organiser vos emails par importance.
🎯 Test pratique :
Observez les suggestions de réponse en bas de vos emails et le tri automatique "Important/Tout" !
NLP Classification Génération de texte
🗺️
Waze
Prédiction
Waze prédit les embouteillages, calcule les itinéraires optimaux en temps réel et détecte automatiquement les incidents de circulation.
🎯 Test pratique :
Lancez un trajet et observez comment l'IA recalcule en temps réel selon le trafic !
Prédiction Analyse de données Géolocalisation
🛒
Amazon
E-commerce IA
Amazon utilise l'IA pour ses recommandations produits, la gestion des stocks, l'optimisation des prix et même la logistique des entrepôts.
🎯 Test pratique :
Regardez "Les clients ayant acheté cet article ont également acheté" - c'est de l'IA en action !
Recommandation Pricing dynamique Logistique

🎮 Exercice : "Est-ce de l'IA ou non ?"

3.3 Démystification : séparer le vrai du marketing

Testez votre capacité à identifier la vraie IA des simples automatisations ou du marketing !

📱 Une application qui vous rappelle de boire de l'eau toutes les 2 heures
🤖
C'est de l'IA
L'app apprend de mes habitudes
⏰
C'est juste une alarme programmée
Pas d'apprentissage, juste des rappels fixes
✅ Correct ! Une simple alarme répétitive n'est pas de l'IA. Pour que ce soit de l'IA, l'app devrait adapter les rappels selon votre activité, la météo, ou vos habitudes de consommation.
🎯 LinkedIn qui vous propose des "Personnes que vous pourriez connaître"
🧠
C'est de l'IA
Analyse de réseaux et patterns de connexions
📞
C'est juste vos contacts téléphoniques
LinkedIn lit votre carnet d'adresses
✅ Correct ! LinkedIn utilise des algorithmes de Machine Learning pour analyser les connexions communes, les entreprises, les localisations et comportements pour prédire qui vous pourriez connaître.
🏠 Un thermostat "intelligent" qui maintient 21°C en permanence
🤖
C'est de l'IA
Il régule automatiquement la température
🌡️
C'est juste un thermostat programmable
Pas d'apprentissage, juste une consigne fixe
✅ Correct ! Maintenir une température fixe n'est pas de l'IA. Un vrai thermostat IA apprendrait vos habitudes, anticiperait vos retours, s'adapterait à la météo et optimiserait la consommation.

📝 Points clés à retenir

Ce que vous avez appris aujourd'hui :

✅ Les 3 familles d'IA :

  • Machine Learning : Apprentissage de patterns dans les données
  • Deep Learning : Réseaux de neurones pour données complexes
  • LLM : Spécialisés dans le langage et la conversation

🦁 Votre Safari IA :

  • Netflix, Spotify, Google Photos utilisent déjà l'IA pour vous
  • L'IA est partout mais souvent invisible
  • Chaque interaction génère des données pour améliorer l'expérience

⚠️ Démystification :

  • Automatisation ≠ Intelligence artificielle
  • Le marketing abuse souvent du terme "IA"
  • La vraie IA apprend et s'adapte continuellement
