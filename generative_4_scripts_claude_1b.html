Transformez une simple idée en script TikTok/YouTube prêt à tourner avec un workflow structuré : Brief → Hook → Structure → CTA → Test A/B.

💪 Exercice pratique guidé

🎯 Contexte et objectifs de ces exercices

Situation : Vous êtes coach business et voulez créer du contenu TikTok/YouTube pour attirer des entrepreneurs débutants. Votre objectif est de démontrer votre expertise tout en créant un lien émotionnel avec votre audience.

Enjeu : Sur les réseaux sociaux, vous avez 3 secondes pour capter l'attention. Votre script doit non seulement informer mais aussi divertir et inciter à l'action, le tout en moins d'une minute.

Méthodologie : Nous utilisons la force de Claude pour créer des scripts naturels et cohérents avec une approche structurée en 5 étapes qui maximise les chances de viralité.

Résultat final : 2 scripts prêts à tourner avec angles différents pour tester lequel fonctionne le mieux avec votre audience.

1
Analyse du concept et définition de l'angle viral

Action : Analysez votre idée de base et définissez l'angle le plus viral possible avec Claude. L'objectif est de transformer un concept banal en accroche irrésistible.

Concept test à utiliser :

Idée de base
"Je veux faire une vidéo pour expliquer pourquoi mon service de coaching business peut aider les entrepreneurs débutants."

Prompt à utiliser :

Prompt 1 - Angle viral
Tu es un expert en contenu viral sur TikTok et YouTube, spécialisé dans les vidéos qui génèrent millions de vues. ANALYSE CETTE IDÉE : "Je veux faire une vidéo pour expliquer pourquoi mon service de coaching business peut aider les entrepreneurs débutants." MISSION : Transforme cette idée en angle viral irrésistible. ANGLES VIRAL À EXPLORER : 1. CONTRE-INTUITIF : "Pourquoi 90% des coachs business donnent de mauvais conseils" 2. HISTOIRE PERSONNELLE : "Comment j'ai échoué 7 fois avant de réussir" 3. MYTHES CASSÉS : "3 mensonges qu'on raconte aux entrepreneurs débutants" 4. DÉMONSTRATION : "Je transforme une idée en 10k€ en 60 secondes" 5. COMPARAISON CHOC : "Entrepreneuriat vs Job : la vérité que personne ne dit" POUR CHAQUE ANGLE : - Hook de 3 secondes (question ou affirmation choc) - Promesse claire (ce que la personne va apprendre) - Niveau de controverse (1-10, 7+ pour viral) - Public cible précis - Potentiel d'engagement estimé OBJECTIF : Trouver l'angle qui fait dire "WOW, je dois voir ça !"

Résultat attendu : 5 angles viraux différents avec hooks, promesses et niveau de controverse. Claude doit identifier l'angle avec le plus fort potentiel viral pour votre audience.

2
Création du hook ultime (3 premières secondes)

Action : Créez plusieurs variations du hook parfait pour les 3 premières secondes. Ces secondes déterminent si votre vidéo sera scrollée ou regardée jusqu'au bout.

Prompt à utiliser :

Prompt 2 - Hook viral
Concentre-toi sur l'angle viral le plus prometteur de l'exercice précédent. MISSION : Crée 5 hooks différents pour les 3 premières secondes. FORMATS DE HOOKS VIRAL : 1. QUESTION CHOC : "Devine combien j'ai perdu en 6 mois ?" 2. AFFIRMATION CONTROVERSE : "Arrête de suivre tes passions" 3. STATISTIQUE SURPRENANTE : "97% des entrepreneurs font cette erreur" 4. HISTOIRE PERSONNELLE : "À 25 ans, j'étais fauché. Aujourd'hui..." 5. DÉFI DIRECT : "Si tu n'arrives pas à faire ça, oublie l'entrepreneuriat" CONTRAINTES TECHNIQUES : - Maximum 8-10 mots - Lisible rapidement à l'écran - Prononçable en 3 secondes max - Crée immédiatement curiosité ou émotion - Fonctionne SANS le son POUR CHAQUE HOOK : - Version écrite (pour sous-titres) - Version parlée (pour voix-off) - Émotion déclenchée (curiosité, peur, espoir, colère) - Score viral estimé (1-10) OBJECTIF : Hook qui stoppe le scroll dans les 0.5 premières secondes.

Résultat attendu : 5 hooks différents avec versions écrite/parlée, émotions déclenchées et scores viraux. Choisissez celui qui génère le plus de curiosité immédiate.

3
Structure narrative engageante (corps du script)

Action : Développez le corps de votre script avec une structure narrative qui maintient l'attention du début à la fin. Utilisez les techniques de storytelling adaptées au format court.

Prompt à utiliser :

Prompt 3 - Structure narrative
Développe le script complet basé sur le meilleur hook de l'exercice précédent. STRUCTURE NARRATIVE VIRAL (30-60 secondes) : INTRO (0-3s) : Hook sélectionné PROBLÈME (3-15s) : Pain point que vit ton audience AGITATION (15-25s) : Amplifier le problème avec exemple concret SOLUTION (25-45s) : Ta méthode/approche unique PREUVE (45-55s) : Résultat rapide ou témoignage CTA (55-60s) : Action simple et claire TECHNIQUES NARRATIVES : - Utilisé "Tu" pour interpeller directement - Transitions fluides entre chaque partie - Langage simple, phrases courtes - Créer mini-cliffhangers ("Mais voici le truc...") - Rythme soutenu, pas de temps mort CONTRAINTES PRODUCTION : - Script lisible à voix haute - Indications visuelles entre parenthèses - Moments de pause marqués - Synchronisation texte/image suggérée OBJECTIF : Script qui maintient 80%+ de l'audience jusqu'à la fin.

Résultat attendu : Script complet 30-60 secondes avec structure narrative claire, transitions fluides et indications de production. Le rythme doit être soutenu sans temps mort.

4
Call-to-action naturel et persuasif

Action : Optimisez la fin de votre script avec un CTA qui convertit sans paraître vendeur. L'objectif est d'inciter à l'action tout en restant dans la valeur.

Prompt à utiliser :

Prompt 4 - CTA optimisé
Optimise les 10 dernières secondes du script avec un CTA qui convertit naturellement. TYPES DE CTA PAR PLATEFORME : TIKTOK : - "Sauvegarde ça pour plus tard" - "Partage à quelqu'un qui en a besoin" - "Follow pour plus de tips comme ça" - "Commente ton plus gros challenge" YOUTUBE SHORTS : - "Abonne-toi si ça t'a aidé" - "Like si tu veux la suite" - "Regarde cette vidéo pour aller plus loin" - "Dis-moi en commentaire ton résultat" LINKEDIN/INSTAGRAM : - "Envoie-moi un DM si tu veux de l'aide" - "Connecte-toi pour plus de contenu" - "Partage ton expérience en commentaire" STRUCTURE CTA NATUREL : 1. TRANSITION : "Maintenant, voici ce que tu peux faire..." 2. ACTION PRINCIPALE : Claire et simple 3. BÉNÉFICE : Pourquoi faire cette action 4. FACILITÉ : "Ça prend 2 secondes" ÉVITER : "Achète maintenant", "Contacte-moi", "Visite mon site" OBJECTIF : CTA qui donne envie d'agir sans pression commerciale.

Résultat attendu : 3-4 options de CTA adaptées à différentes plateformes, naturels et non-intrusifs. Le CTA doit prolonger l'engagement plutôt que vendre directement.

5
Création de la version A/B alternative

Action : Créez une version alternative de votre script pour tester l'efficacité. Changez l'angle ou l'approche pour identifier quelle version résonne le mieux avec votre audience.

Prompt à utiliser :

Prompt 5 - Version A/B
Crée une version alternative du script en changeant l'approche narrative. VERSION A (existante) : [Insérer le script créé précédemment] VERSION B À CRÉER : Si Version A = Problème → Solution → Preuve Alors Version B = Histoire → Leçon → Application ALTERNATIVES NARRATIVES : 1. AVANT/APRÈS : "J'étais X, maintenant je suis Y" 2. ERREURS → CORRECTIONS : "J'ai fait ces 3 erreurs, voici comment les éviter" 3. MÉTHODE PAS-À-PAS : "Voici exactement comment faire" 4. COMPARAISON : "Méthode classique vs ma méthode" 5. DÉMONSTRATION : "Je vais te prouver en temps réel" ÉLÉMENTS À TESTER : - Hook différent (même message, angle opposé) - Ton différent (sérieux vs décontracté) - Structure différente (chronologique vs thématique) - CTA différent (action vs réflexion) CONTRAINTES : - Même durée (30-60s) - Même objectif final - Même audience cible - Message cohérent mais approche différente OBJECTIF : 2 scripts prêts pour test A/B réel.

Résultat attendu : Version B complète avec approche narrative différente, même durée et même objectif. Vous aurez 2 scripts prêts pour tester lequel fonctionne le mieux.

📊 Validation des résultats

📏 Checklist de qualité finale :

Temps total de création ~25 min
Durée des scripts finaux 30-60 sec
Versions prêtes pour test 2 scripts A/B
Hook viral optimisé 3 premières sec

🏆 Éléments à vérifier :

  • ✅ Hook viral : Stoppe le scroll dans les 3 premières secondes
  • ✅ Structure fluide : Transitions naturelles sans temps mort
  • ✅ Durée optimale : 30-60 secondes chrono à voix haute
  • ✅ CTA naturel : Incite à l'action sans pression commerciale
  • ✅ Versions A/B : Angles différents pour tester l'efficacité
  • ✅ Prêt production : Indications visuelles et pauses marquées

💡 Comment tester vos scripts :

  • Chronométrage : Lisez à voix haute, 30-60s max
  • Test d'écoute : Enregistrez-vous, ça sonne naturel ?
  • Test de compréhension : Quelqu'un comprend sans contexte ?
  • Test émotionnel : Génère curiosité, empathie ou action ?
  • Test visuel : Fonctionne-t-il même sans le son ?
🚀 Prochaine étape

Action immédiate : Tournez les 2 versions cette semaine. Publiez Version A puis Version B quelques jours après. Analysez les métriques (vues, engagement, partages).

Optimisation : Gardez le format qui marche le mieux et créez 5 variations sur le même modèle !
